onEsc
directive in module ngPortalApp

Description

Runs an expression on escape keyup.

Usage

as attribute
<ANY on-esc="{Expression}">
   ...
</ANY>
as class
<ANY class="on-esc: {Expression};">
   ...
</ANY>

Parameters

ParamTypeDetails
onEscExpression

An expression to execute once (and only once) when the escape button is pressed.
