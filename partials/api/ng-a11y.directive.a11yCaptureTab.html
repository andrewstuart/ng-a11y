a11yCaptureTab
directive in module ng-a11y

Description

When present on the page, the tabbable elements between the selectors will have tab input captured.

Usage

as attribute
<ANY a11y-capture-tab
     capture-immediate="{Expression}"
     force-capture="{Expression}">
   ...
</ANY>

Parameters

ParamTypeDetails
captureImmediateExpression

If the expression value is truthy, then upon the directive loading, focus will be triggered.

forceCaptureExpression

If the expression value is truthy, then each tab anywhere within the document will trigger a focus to the first element.
