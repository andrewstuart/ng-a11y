a11yModal
directive in module ng-a11y

Description

Appends a global hider with class 'a11yModal-hider' (you must style it yourself, or use some preexisting style library; see example) and shows the content inside the <a11yModal> element.

This a11yModal module will be A11Y-compliant if you properly implement an expression in the modalHide attribute which will cause shown to evaluate to false.

Usage

as element:
<a11y-modal
       shown="{Boolean}"
       [modal-hide]="{Function}"
       [modal-show]="{Function}">
</a11y-modal>
as attribute
<ANY a11y-modal
     shown="{Boolean}"
     [modal-hide]="{Function}"
     [modal-show]="{Function}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
shownBoolean

Whether or not the a11yModal dialog should be hidden via the "hidden" class. Implementation of this class is up to the consumer.

[modalHide]Function

An optional method to be called any time the a11yModal dialog is hidden by the user interacting with dialog-specific elements (e.g. clicking the backdrop, or "esc" key)

[modalShow]Function

An optional method to be called any time the a11yModal dialog is unhidden

Methods

  • hide()

    hide is a method to hide the a11yModal
