a11yEsc
directive in module ng-a11y

Description

Runs an expression on escape keyup.

Usage

as attribute
<ANY a11y-esc
     a11yon-esc="{Expression}">
   ...
</ANY>
as class
<ANY class="a11y-esc a11yon-esc: {Expression};">
   ...
</ANY>

Parameters

ParamTypeDetails
a11yonEscExpression

An expression to execute once (and only once) when the escape button is pressed.
